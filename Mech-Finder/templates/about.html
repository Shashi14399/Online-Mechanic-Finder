About Us


This is an online mechanic finding website created as part of the project for the WebTech-II course of 2019-20 by Srivatsa Vantmuri, Saransh Gupta and Shashidhar R. We have tried to implement the basic functionality of a mechanic finding website in which the customers can request a mechanic at their location and aspiring mechanics can find a job. The front-end uses different CSS and Bootstrap styling features to make the website more appealing and also integrates javascript to propagate this.The backend uses MYSql database to store appropriate information that the webpage requires, this data is accessed with the help of FLASK which excecutes them on the server side and thus integrating all the HTML functionality with that of CSS and JS to produce a website capable of implementing some of the features of a mechanic finding website on a smaller scale.
